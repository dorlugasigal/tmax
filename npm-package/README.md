# tmax-terminal

Powerful multi-terminal app with tiling, floating panels, and keyboard-driven workflow.

## Install

```bash
npm install -g tmax-terminal
```

Then run:

```bash
tmax
```

## One-time use

```bash
npx tmax-terminal
```

## What it does

This package downloads the correct tmax binary for your platform (Windows, macOS, Linux — x64 and ARM64) and provides a `tmax` command to launch it.
