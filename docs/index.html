Features

▦

Tiling Layouts

Split terminals horizontally, vertically, or float them. Grid mode arranges everything automatically.

✨

AI Session Management

Monitor Claude Code and GitHub Copilot sessions. Resume, rename, and organize with lifecycle tabs.

📁

File Explorer

Browse files, preview code inline, navigate with breadcrumbs. Works with WSL filesystems.

🎨

Tab Groups

Group related terminals with colors, collapsible headers, and drag between groups.

⌨

Keyboard-Driven

Command palette, pane hints, fully customizable shortcuts. Works with Cmd on macOS.

🐧

WSL Integration

Discover AI sessions from WSL distros, resume in the right environment.

📄

Diff Review

Built-in code diff overlay with file tree and inline annotations.

🌈

Appearance

System font picker, Windows 11 Mica/Acrylic, 10+ themes, dark title bar.

🔄

Auto Updates

Seamless updates on Windows, macOS, and Linux. See release notes before updating.

Key Shortcuts

ShortcutAction
Ctrl+Shift+PCommand palette
Ctrl+Shift+NNew terminal
Ctrl+Alt+ArrowSplit in direction
Shift+ArrowMove focus between panes
Ctrl+Shift+FToggle Focus / Grid mode
Ctrl+Shift+CAI Sessions panel
Ctrl+Shift+XFile Explorer
Ctrl+Shift+KJump to AI prompt
Ctrl+Shift+BHide / show tab bar
Ctrl+,Settings

On macOS, Ctrl is replaced with Cmd (⌘). All shortcuts are customizable.
